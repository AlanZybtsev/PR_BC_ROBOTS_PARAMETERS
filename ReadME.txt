Перед новым прогоном скрипта убедиться:

1. В отсутствии папки "PARAMETERS"
2. Исходные файлы с параметрами роботов находятся в директории ".//ROB_PAR//{номер робота}//BRUSHES.INI"
_____________________

Скрипт к выполнению находится в файле main.py. 
Предназначен для перевода параметров нанесения вторичного грунта, базисной эмали из исходных текстовых файлов, 
выкачанных непосредственно с роботов окраски завода "...", в файлы excel (формат xlsx). 

После выполнения скрипта excel файлы будут находиться в директории ".//PARAMETERS//{модель}//{модель} - 
{дата прогона скрипта}.xlsx".
Название листа в файле - номер циркуляционной систем
_____________________


Облегчается воспрятие сжатой, конкретизированной информации. Печать полученных документов возможна после 
нес ложного форматирования excel файлов.
